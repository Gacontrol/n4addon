# Wiresheet Add-ons für Home Assistant

[![GitHub Release][releases-shield]][releases]
[![License][license-shield]](LICENSE)

Ein visueller Flow-Editor für Home Assistant - Programmiere mit Drag & Drop Bausteinen wie in Niagara Wiresheet.

## Installation

1. Klicke auf diesen Button, um das Repository zu Home Assistant hinzuzufügen:

   [![Add Repository to Home Assistant][repository-badge]][repository-url]

   Oder füge diese URL manuell hinzu:
   ```
   https://github.com/Gacontrol/n4addon
   ```

2. Navigiere zu **Einstellungen** → **Add-ons** → **ADD-ON STORE**
3. Suche nach "Wiresheet Flow Editor"
4. Klicke auf das Addon und dann auf **INSTALLIEREN**
5. Starte das Addon
6. Klicke auf **WEB UI ÖFFNEN** oder nutze die Sidebar

## Add-ons in diesem Repository

### Wiresheet Flow Editor

![Wiresheet Logo][wiresheet-icon]

Visueller Flow-Editor für Home Assistant. Erstelle Smart Home Automatisierungen mit grafischen Bausteinen.

**Features:**
- Drag & Drop Interface
- 10+ verschiedene Bausteine (Sensoren, Aktoren, Logik, Trigger)
- Direkte Home Assistant Integration
- Export/Import von Flows
- Automatisches Speichern
